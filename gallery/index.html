formura-egison ギャラリー 検証済み応用 18 例

Egison のテンソル添字記法・計算機代数(CAS)で書いた連続系の物理を、 substitute による座標シフトで離散化し、Formura(分散並列ステンシル DSL)の コードとして出力して実行した結果です。各カードには支配方程式・実際に書いた Egison ソース(物理部分)・ そこから自動生成された Formura コード(.fmr)・実行結果の可視化・検証数値を載せています。 すべて make all で再現でき、画像は gallery/gen.sh → gallery/tools/render.py(外部ライブラリ不要)で再生成できます。

基礎と横断機能

拡散方程式

∂tu = κ∇²u
Egison — 物理はこの1行
def uNext := u + dt * kappa * lap u
生成された Formura(.fmr) 
u' = u[i,j,k] + (-2)*dt*kappa*u[i,j,k]/(dz**2) + dt*kappa*u[i,j,k-1]/(dz**2)
   + dt*kappa*u[i,j,k+1]/(dz**2) + ...          # 3軸7点ステンシルに自動展開
z 中央断面: t=0 のガウス塊(左)が等方に拡散(右、100 step)
100³格子・temporal blocking 5 で 0.19 秒。質量保存 6.4e-13(機械精度)。

4次精度スキーム自動導出(taylorStencil)

Σk ck km/m! = δm,2 (m=0..4)  ⟹  c = (−1/12, 4/3, −5/2, 4/3, −1/12)
係数はソースのどこにも書かれていない — CAS が厳密有理数で連立を解く
Egison — Taylor 条件を解いてステンシルを合成
def taylorStencil (m: Integer) (ks: [MathValue]) : [MathValue] :=
  gaussSolve (map (\mm -> map (\k -> k ^ mm / factI mm) ks ++ [if mm = m then 1 else 0])
                  (between 0 (length ks - 1)))

def lap4 (f: MathValue) : MathValue := sum (map (\a -> dTaylor 2 ks5 a f) [1, 2, 3])
def uNext := u + dt * kappa * lap4 u
生成された Formura(.fmr) 
# CAS-derived 4th-order coefficients c(-2..2): [-1 / 12, 4 / 3, -5 / 2, 4 / 3, -1 / 12]
u' = u[i,j,k] + (-5)*dt*kappa*u[i,j,k]/(2*dz**2) + 4*dt*kappa*u[i,j,k-1]/(3*dz**2)
   + (-1)*dt*kappa*u[i,j,k-2]/(12*dz**2) + ...
ラプラシアンの離散シンボル誤差 |λ(k)+k²|: 導出4次(青)は2次(灰)より約2桁下
単一モード振幅比 = 厳密離散シンボル (1+λ₄dt)ⁿ に 4.4e-16 で一致、 残差は4次理論値 k⁶h⁴/90 の 98.6%。

Dirichlet 壁の拡散(fork の境界条件)

∂tu = κ∇²u,  u|壁 = 0
λ = −(4κ/h²) sin²(π/2(N+1)) — ゴースト壁つき離散固有値
Egison + yaml — 物理は周期版と同一、壁は yaml の1行
def uNext := u + dt * kappa * lap u          -- Egison 側は不変

boundary: [fixed 0.0, periodic, periodic]    # yaml(fork 拡張)
冷壁間の基本固有モードが形を保ったまま減衰(t=0 → 0.2)
厳密減衰 (1+λdt)ⁿ を 6.8e-14 で再現、モード純度 1.2e-15。

波動(スタガード格子 = yeeRef 機構)

Maxwell(collocated)

∂tE = ∇×B,  ∂tB = −∇×E  (c=1)
Egison — curl は ε 縮約で一度だけ定義、物理は2行
def curl (X: Vector MathValue) : Vector MathValue :=
  withSymbols [i, j, k] (ε 3)~i~j~k . (dGrad X)_j_k    -- Levi-Civita 縮約

def En : Vector MathValue := E + dt * curl B
def Bn : Vector MathValue := B - dt * curl En          -- symplectic Euler
生成された Formura(.fmr) 
Ex' = Ex[i,j,k] + (-1)*Bz[i,j-1,k]*dt/(2*dy) + Bz[i,j+1,k]*dt/(2*dy)
    + By[i,j,k-1]*dt/(2*dz) + (-1)*By[i,j,k+1]*dt/(2*dz)
...                                            # 6成分の更新式に自動展開
Ey パルスの伝播(修正版コンパイラで +9.9 セル/理想 +10)
エネルギードリフト 4.8e-5。2ランク実 MPI でランク境界を完全通過。

Yee-FDTD(スタガード格子)

同上 — ただし E は辺・B は面: σE=(½,0,0)…, σB=(0,½,½)…
Egison — 配置オフセット σ を宣言すると教科書の FDTD が出る
def sE := [[1 / 2, 0, 0], [0, 1 / 2, 0], [0, 0, 1 / 2]]        -- E: 辺
def sB := [[0, 1 / 2, 1 / 2], [1 / 2, 0, 1 / 2], [1 / 2, 1 / 2, 0]]  -- B: 面

def eRhs i1 e := e + dt * curlYee i1 (nth i1 sE) bFlds
def bRhs i1 b := b - dt * curlYee i1 (nth i1 sB) eNFlds    -- leapfrog
生成された Formura(.fmr) 
Ex' = Ex[i,j,k] + Bz[i,j,k]*dt/(dy) + (-1)*Bz[i,j-1,k]*dt/(dy)
    + (-1)*By[i,j,k]*dt/(dz) + By[i,j,k-1]*dt/(dz)   # 半セル差分が整数幅1に落ちる
dt=0.5dx で +49.9 セル伝播(理想 +50)
div B ≡ 0(機械精度)・エネルギー 0.10%・TB あり/なしビット一致。

線形音響(p–v)

∂tp = −K∇·v,  ∂tv = −∇p/ρ
Egison — Virieux のスカラー縮約(v を先に、p は更新済み v' で)
fmrEq "vx'" (vx - (dt / rho0) * dYee 1 sVx (p, s0))
fmrEq "p'"  (p - dt * kap * divv')                 -- divv' は v' の dYee 和
生成された Formura(.fmr) 
vx' = vx[i,j,k] + dt*p[i,j,k]/(dx*rho0) + (-1)*dt*p[i+1,j,k]/(dx*rho0)
インピーダンス整合パルス(p=Zv)が反射なく右進
実測音速 0.9957(厳密 1)・エネルギー 1.6e-4・横速度 =0 厳密。

弾性波(Virieux スタガード格子)

ρ∂tvi = ∂jσij,   ∂tσij = λδij∇·v + μ(∂ivj+∂jvi)
Egison — 速度3+応力6場、全微分が yeeRef の半セル差分
fmrEq "vx'"  (vx + (dt / rho0) * (dYee 1 sVx (sxx, s0)
                + dYee 2 sVx (sxy, sXY) + dYee 3 sVx (sxz, sXZ)))
fmrEq "sxx'" (sxx + dt * (la * divv' + 2 * mu * dYee 1 s0 (vx', sVx)))
fmrEq "sxy'" (sxy + dt * mu * (dYee 1 sXY (vy', sVy) + dYee 2 sXY (vx', sVx)))
生成された Formura(.fmr) 
vx' = vx[i,j,k] + dt*sxz[i,j,k]/(dz*rho0) + (-1)*dt*sxz[i,j,k-1]/(dz*rho0)
    + dt*sxy[i,j,k]/(dy*rho0) + (-1)*dt*sxy[i,j-1,k]/(dy*rho0) + ...
同時発射した P パルス(vx)と S パルス(vy)が速度差で分離
1回の実行で vp=1.990/2・vs=0.995/1 を実測、エネルギー 3.4e-4。

非線形 Klein–Gordon(φ⁴ キンク)

∂ttφ = ∂xxφ + φ − φ³
二重井戸 V=(φ²−1)²/4、キンク = tanh((x−x₀)/√2)
Egison — leapfrog(kick–drift、drift は更新済み w' を参照)
def wNext   := w + dt * (dC2 1 phi + phi - phi * phi * phi)   -- kick
def phiNext := phi + dt * w'                                  -- drift
生成された Formura(.fmr) 
w'   = (-1)*dt*phi[i,j,k]**3 + dt*phi[i,j,k] + w[i,j,k] + (-2)*dt*phi[i,j,k]/(dx**2)
     + dt*phi[i-1,j,k]/(dx**2) + dt*phi[i+1,j,k]/(dx**2)
phi' = dt*w'[i,j,k] + phi[i,j,k]
v=±0.2 にブーストした kink–antikink 対が接近(t=0→20→40)
実測速度 ±0.1993・E=2γE_kink に偏差 0.11%・ シンプレクティックでドリフト 3.3e-7。

流体・保存則系

Burgers 方程式

∂tu + u∂xu = ν∇²u
Egison — 非線形移流もそのまま書く
def dUdt := nu * lap u - u * dC 1 u
生成された Formura(.fmr) 
u' = dt*u[i-1,j,k]*u[i,j,k]/(2*dx) + (-1)*dt*u[i+1,j,k]*u[i,j,k]/(2*dx)
   + u[i,j,k] + (-2)*dt*nu*u[i,j,k]/(dz**2) + ...
t=0.5 の数値解と Cole–Hopf 厳密解(赤破線)が重なる
厳密解と max 誤差 3.5e-5(離散化誤差オーダー)。

浅水方程式

∂th + ∇·(hu) = 0,   ∂t(hu) + ∂x(hu² + gh²/2) + ∂y(huv) = 0 …
Egison — 保存形+粘性を upd に一度だけ書く
def upd q fx fy := q - dt * (dC 1 fx + dC 2 fy) + dt * lam * (dC2 1 q + dC2 2 q)

fmrEq "fmxx" (mx * mx / h + g * h * h / 2)   -- 運動量流束(中間場)
fmrEq "h'"  (upd h mx my)
fmrEq "mx'" (upd mx fmxx fmxy)
生成された Formura(.fmr) 
fmxx = g*h[i,j,k]**2/(2) + mx[i,j,k]**2/(h[i,j,k])
h' = h[i,j,k] + dt*my[i,j-1,k]/(2*dy) + (-1)*dt*my[i,j+1,k]/(2*dy)
   + dt*mx[i-1,j,k]/(2*dx) + ...
静水面の 1% バンプが左右の重力波に分裂(t=20)
実測波速 0.9989(厳密 √(gh₀)=1)・質量 3.7e-14・max|my|=0 ビット厳密。

圧縮性 Euler: Sod 衝撃管

∂tρ + ∂x(ρu) = 0,  ∂t(ρu) + ∂x(ρu²+p) = 0,  ∂tE + ∂x((E+p)u) = 0
p = (γ−1)(E − ρu²/2)、γ=1.4
Egison — 圧力と流束を中間場に、更新は upd
fmrEq "pr" ((2 / 5) * (en - mx * mx / (2 * rho)))    -- (γ−1) = 2/5
fmrEq "fm" (mx * mx / rho + pr)
fmrEq "fe" ((en + pr) * mx / rho)
fmrEq "rho'" (upd rho mx)
生成された Formura(.fmr) 
pr = (-1)*mx[i,j,k]**2/(5*rho[i,j,k]) + 2*en[i,j,k]/(5)
fm = pr[i,j,k] + mx[i,j,k]**2/(rho[i,j,k])
衝撃波・接触不連続・膨張扇の3波構造 vs 厳密 Riemann 解(破線)
L1(ρ) = 0.0255・質量/運動量/エネルギー保存 ~1e-14・正値性維持。

理想 MHD: Orszag–Tang 渦

保存形8変数(ρ, ρv, B, E)+全圧 pt = p + B²/2
Egison — 流束19本+更新8本(手書き HLLD 283 行に対し)
def upd u dfs := u - dt * sum (map (\(d, f) -> d1 d f) dfs) + dt * lam * visc u

fmrEq "fmx1" (mx * mx / rho - bx * bx + pt)      -- 運動量流束(9本の例)
fmrEq "fbx2" ((my * bx - mx * by) / rho)          -- induction(6本の例)
fmrEq "rho'" (upd rho [(1, mx), (2, my), (3, mz)])
...
生成された Formura(.fmr) 
fmx1 = (-1)*bx[i,j,k]**2 + pt[i,j,k] + mx[i,j,k]**2/(rho[i,j,k])
rho' = (-1)*dt*lam*rho[i,j,k]/(dz) + dt*lam*rho[i,j,k-1]/(2*dz) + ...
t=0.5 の密度場 — 特徴的な OT 渦構造

格子ボルツマン D3Q19

fa(x+ca, t+1) = fa + (feqa − fa)/τ
feqa = waρ(1 + 3c·u + 9⁄2(c·u)² − 3⁄2u²)
Egison — 57本の式を速度集合への map だけで生成
def collLines := map (\(w, c, a) -> fmrEq (S.concat ["g", show a])
    (nth (a + 1) fs + (feq w c rho ux uy uz - nth (a + 1) fs) / tau)) trip
def streamLines := map (\(w, c, a) ->
    fmrEq (S.concat ["f", show a, "'"]) (stream c (nth (a + 1) gs))) trip

g3 = (-1)*rho[i,j,k]*uz[i,j,k]**2/(12*tau) + rho[i,j,k]*uy[i,j,k]**2/(6*tau) + ...
f3'  = g3[i,j-1,k]                       # 19方向それぞれ異なるシフト
f18' = g18[i,j+1,k-1]

Kuramoto–Sivashinsky(時空カオス)

∂tu = −(u²/2)x − uxx − uxxxx
Egison — 4階微分は中間場 w=∂²u の2段(袖幅1×2)
fmrEq "w"  (dC2 1 u)
fmrEq "u'" (u - dt * (dC 1 (u * u / 2) + w + dC2 1 w))
生成された Formura(.fmr) 
w  = (-2)*u[i,j,k]/(dx**2) + u[i-1,j,k]/(dx**2) + u[i+1,j,k]/(dx**2)
u' = (-1)*dt*w[i,j,k] + dt*u[i-1,j,k]**2/(4*dx) + (-1)*dt*u[i+1,j,k]**2/(4*dx)
   + u[i,j,k] + 2*dt*w[i,j,k]/(dx**2) + (-1)*dt*w[i-1,j,k]/(dx**2) + ...
時空図(横=x、縦=時間 t:0→90): セル構造が分裂・合体するカオス
Lyapunov 内(t=5)で参照実装と 3.6e-14・Σu ドリフト 5.4e-12(60万 step)。

パターン形成・材料

Pearson 反応拡散(Formura 論文の看板再現)

∂tU = Fu(1−U) − FeUV² + Du∇²U
∂tV = FeUV² − FvV + Dv∇²V
Egison — FHPC'16 Listing 1 と同じ方程式・パラメタ
def dUdt := Fu * (1 - U) - Fe * U * V^2 + Du * lap U
def dVdt := Fe * U * V^2 - Fv * V + Dv * lap V
生成された Formura(.fmr) 
U' = (-1)*Fe*U[i,j,k]*V[i,j,k]**2*dt + (-1)*Fu*U[i,j,k]*dt + Fu*dt
   + U[i,j,k] + (-2)*Du*U[i,j,k]*dt/(dz**2) + ...
V 場の断面(20,000 step): 種スポットがリング状に自己複製 — 論文 Fig.7 と同じ Gray–Scott 型ダイナミクス
64³格子(dt=200s)。40,000 step まで回すとスポットが全域に増殖。物理の記述は2行。

Cahn–Hilliard(スピノーダル分解)

μ = c³ − c − κ∇²c,  ∂tc = M∇²μ
Egison — 4階微分を中間場 μ の2段構成で
def muDef := c^3 - c - kap * lap c
def dcdt  := mob * lap mu
生成された Formura(.fmr) 
mu = c[i,j,k]**3 + (-1)*c[i,j,k] + 2*c[i,j,k]*kap/(dz**2) + ...
c' = (-2)*dt*mob*mu[i,j,k]/(dz**2) + dt*mob*mu[i,j,k-1]/(dz**2) + ...
低波数の正弦ゆらぎから相分離したラメラ(縞)状二相構造(25,000 step、c∈[−0.95,0.96])
質量(reduces 経由)12桁保存・自由エネルギー単調減少。

TDGL 超伝導(量子化渦)

∂tψ = ψ − |ψ|²ψ + ∇²ψ  (ψ = a + ib)

def psi2 := a^2 + b^2

幾何(CAS ならではの看板)

計量つき拡散(トーラス上の Laplace–Beltrami)

∂tu = (1/√g) ∂i(√g gij ∂ju)
トーラス R=2, r=1: √g = 2+cos θ, gθθ=1, gφφ=1/(2+cos θ)²
Egison — 計量は1行、係数展開と init 生成は CAS の仕事
def sqg   := 2 + cos x        -- √g(ユーザが書く計量情報はこれだけ)
def acoef := sqg              -- √g g^θθ
def bcoef := 1 / sqg          -- √g g^φφ

fmrInit "ca" (substitute [(x, x + hx / 2)] acoef)   -- 半セル位置で評価
fmrEq "f1" (ca * dYee 1 sH1 (u, s0))                -- 流束(半セル)
fmrEq "u'" (u + dt * (dYee 1 s0 (f1, sH1) + dYee 2 s0 (f2, sH2)) / sg)
生成された Formura(.fmr)— init の係数式まで自動 
ca[i,j,k] = cos((i*dx) + dx/(2)) + 2          # CAS が記号評価した係数場
cb[i,j,k] = (1)/(cos((i*dx)) + 2)
f1 = (-1)*ca[i,j,k]*u[i,j,k]/(dx) + ca[i,j,k]*u[i+1,j,k]/(dx)
u' = u[i,j,k] + dt*f2[i,j,k]/(dy*sg[i,j,k]) + (-1)*dt*f2[i,j-1,k]/(dy*sg[i,j,k]) + ...
u(θ,φ): 原点の塊(四隅=周期像)が計量の非一様性を感じつつ拡散(左 t=0 → 右 t=3000)
独立 C 参照と 3.3e-16(全桁一致)・計量重みつき熱量 Σ√g·u 保存 2.5e-15・ 最大値原理成立。
