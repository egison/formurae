formura-egison ギャラリー 検証済み応用 18 例

Egison のテンソル添字記法・計算機代数(CAS)で書いた連続系の物理を、 substitute による座標シフトで離散化し、Formura(分散並列ステンシル DSL)の コードとして出力して実行した結果です。各例は解析解・独立参照実装・保存則・厳密離散固有値の いずれかで定量検証されており、make all で全例が再現できます (画像は gallery/gen.sh → gallery/tools/render.py で再生成、外部ライブラリ不要)。

基礎と横断機能

拡散方程式

∂t u = κ∇²u — 物理は u + dt*kappa*lap u の1行
z 中央断面: t=0 のガウス塊(左)が等方に拡散(右、100 step)
100³格子・temporal blocking 5 で 0.19 秒。質量保存 6.4e-13(機械精度)。

4次精度スキーム自動導出(taylorStencil)

係数 (−1/12, 4/3, −5/2, 4/3, −1/12) はソースに無く、Taylor 条件の連立を CAS が厳密有理数で解く
ラプラシアンの離散シンボル誤差 |λ(k)+k²|: 導出4次(青)は2次(灰)より約2桁下
単一モード振幅比 = 厳密離散シンボル (1+λ₄dt)ⁿ に 4.4e-16 で一致、 残差は4次理論値 k⁶h⁴/90 の 98.6%。

Dirichlet 壁の拡散(fork の境界条件)

yaml に boundary: [fixed 0.0, periodic, periodic] — 物理は周期版と同じ1行
冷壁間の基本固有モードが形を保ったまま減衰(t=0 → 0.2)
ゴースト壁つき離散固有値の厳密減衰 (1+λdt)ⁿ を 6.8e-14 で再現、 モード純度 1.2e-15。

波動(スタガード格子 = yeeRef 機構)

Maxwell(collocated)

∂t E = c²∇×B、∂t B = −∇×E — ε 縮約 curl の2行から6式を自動展開
Ey パルスの伝播(修正版コンパイラで +9.9 セル/理想 +10)
エネルギードリフト 4.8e-5。2ランク実 MPI でランク境界を完全通過。

Yee-FDTD(スタガード格子)

E=辺・B=面の配置オフセット σ∈{0,½}³ を宣言 → 教科書どおりの FDTD を生成
dt=0.5dx で +49.9 セル伝播(理想 +50)
div B ≡ 0(機械精度)・エネルギー 0.10%・TB あり/なしビット一致。

線形音響(p–v)

∂t p = −K∇·v、∂t v = −∇p/ρ — Virieux のスカラー縮約
インピーダンス整合パルス(p=Zv)が反射なく右進
実測音速 0.9957(厳密 1)・エネルギー 1.6e-4・横速度 =0 厳密。

弾性波(Virieux スタガード格子)

ρ∂t v = ∇·σ、∂t σ = λ(∇·v)I + μ(∇v+∇vᵀ) — 速度3+応力6場
同時発射した P パルス(vx)と S パルス(vy)が速度差で分離
1回の実行で vp=1.990/2・vs=0.995/1 を実測、エネルギー 3.4e-4。

非線形 Klein–Gordon(φ⁴ キンク)

∂tt φ = ∇²φ + φ − φ³ — leapfrog(kick–drift)2場
v=±0.2 にブーストした kink–antikink 対が接近(t=0→20→40)
実測速度 ±0.1993・E=2γE_kink に偏差 0.11%・ シンプレクティックでドリフト 3.3e-7。

流体・保存則系

Burgers 方程式

∂t u + u∂x u = ν∇²u — 非線形項生成の機械検証
t=0.5 の数値解と Cole–Hopf 厳密解(赤破線)が重なる
厳密解と max 誤差 3.5e-5(離散化誤差オーダー)。

浅水方程式

∂t h + ∇·(hu) = 0 ほか保存形 — 全項が流束形式
静水面の 1% バンプが左右の重力波に分裂(t=20)
実測波速 0.9989(厳密 √(gh₀)=1)・質量 3.7e-14・max|my|=0 ビット厳密。

圧縮性 Euler: Sod 衝撃管

保存形3変数+Lax–Friedrichs 粘性(周期格子用の二重ダイアフラム)
衝撃波・接触不連続・膨張扇の3波構造 vs 厳密 Riemann 解(破線)
L1(ρ) = 0.0255・質量/運動量/エネルギー保存 ~1e-14・正値性維持。

理想 MHD: Orszag–Tang 渦

保存形8変数+Rusanov 流束を中間流束場19本として生成
t=0.5 の密度場 — 特徴的な OT 渦構造
8保存量ドリフト ~1e-12・divB=1.2e-14・正値性維持。 物理記述 = 流束19行+更新8行(手書き HLLD 283 行に対し)。

格子ボルツマン D3Q19

f_a(x+c_a, t+1) = f_a + (feq_a − f_a)/τ — 57式を速度集合への map で生成
剪断波の粘性減衰(1000 step で振幅 38.1%)
実測 ν=0.10010(BGK 厳密 (τ−½)/3=0.1)・質量 4.8e-14。 19出力が全て異なるオフセット = 混在オフセット修正の回帰ショーケース。

Kuramoto–Sivashinsky(時空カオス)

∂t u = −(u²/2)ₓ − uₓₓ − uₓₓₓₓ — 4階は中間場 w=∂²u の2段構成
時空図(横=x、縦=時間 t:0→90): セル構造が分裂・合体するカオス
Lyapunov 内(t=5)で参照実装と 3.6e-14・Σu ドリフト 5.4e-12(60万 step)。

パターン形成・材料

Pearson 反応拡散(Formura 論文の看板再現)

FHPC'16 と同一の方程式・パラメタ(菌根菌 mycorrhiza モデル)
V 場の断面(20,000 step): 種スポットがリング状に自己複製 — 論文 Fig.7 と同じ Gray–Scott 型ダイナミクス
64³格子(dt=200s)。40,000 step まで回すとスポットが全域に増殖。物理の記述は2行。

Cahn–Hilliard(スピノーダル分解)

μ = c³−c−κ∇²c、∂t c = M∇²μ — 4階微分を中間場 μ の2段で
低波数の正弦ゆらぎから相分離したラメラ(縞)状二相構造(25,000 step、c∈[−0.95,0.96])
質量(reduces 経由)12桁保存・自由エネルギー単調減少。

TDGL 超伝導(量子化渦)

∂t ψ = ψ − |ψ|²ψ + ∇²ψ(実2場)
|ψ|² 場: 暗い点が量子化渦芯(min |ψ|²=0.004)
バルク |ψ|²=0.978 に飽和・渦芯 48 セル自発形成。3.6 秒。

幾何(CAS ならではの看板)

計量つき拡散(トーラス上の Laplace–Beltrami)

∂t u = (1/√g)∂ᵢ(√g gⁱʲ∂ⱼu) — √g, gⁱʲ は計量から CAS が記号導出し init まで生成
u(θ,φ): 初期塊(左)が計量の非一様性を感じつつ拡散(右)。 横=θ(外周角)で √g=2+cosθ が左右非対称を生む
独立 C 参照と 3.3e-16(全桁一致)・計量重みつき熱量 Σ√g·u 保存 2.5e-15・ 最大値原理成立。生成 init 例: ca[i,j,k] = cos((i*dx)+dx/(2))+2
