1. 要点 — 「assert を消す」だけでは成立しない

現在 Formurae は、metric scale [...] / embedding [...] で宣言された幾何の誘導計量が 記号的に直交(非対角成分がすべて 0 に簡約)であることを pre-fec の CAS 検査で要求し、 そうでなければ reject する。この検査は単なるガードではなく、 下流全体が「計量 = scale factor の対角行列」という表現に依存している: volume = ∏hᵢ、Hodge 係数 = V/∏hᵢ²、lb 係数 = V/hₐ²。 検査だけ外すと、これらが非対角成分を黙って無視した誤った数値を出す。

したがって「直交性要求の削除」の実体は、 一般計量を正しく扱う経路の新設 + 直交性検査の役割を gate から分類器へ変更することである。 論文(formurae-paper/main.tex の limitations)も必要 3 要素を明言している: (1) covariant operator の仕様確定、(2) cross term の placement 規則、 (3) 新規または改訂した opaque discrete contract。 長期の受け皿は design/20260713 §9.3 の typed operator graph (weighted-adjoint-d primitive)である。

4. staggered DEC と一般計量 Hodge の衝突

staggered DEC の核心的な利点 — 離散 Hodge 星が対角、つまり 「同一格子点の 1 つの値に係数を掛けるだけ」の演算になること — は、 計量の直交性の帰結そのものである。 一般計量でも staggered 格子は数学的には組めるが、Hodge が 「異なる配置に住む成分を混ぜる平均化行列」に変わり、点ごとの演算ではなくなる。

4.1 なぜ直交だと点ごとで済むか

Yee 型の staggered 配置では k-form の成分ごとに住所が違う (下図: α₁ は x 辺の中点、α₂ は y 辺の中点)。直交計量の Hodge は

(⋆α)_J = ± (V / ∏_{i∈I} hᵢ²) · α_I,   I = J の補集合

で、各ターゲット成分がちょうど 1 つのソース成分にしか依存しない。 さらに Yee の双対性により primal の I-セルと dual の Ic-セルは中心が同一点。 つまり直交 ⋆ は「同じ場所の 1 値 × 係数場」で完結し、補間ゼロ・厳密に可逆。 実装もこれに正確に対応している(makeHodgeCoefficient はソース placement での 1 点サンプル、係数場の role は basis 1 つに紐づく OrthogonalHodgeCoefficientRole)。

4.2 一般計量で壊れる箇所

2 次元の 1-form で書くと (⋆α)₂ = √g (g¹¹α₁ + g¹²α₂)。 直交なら g¹² = 0 で右辺は同一点の α₁ だけ。g¹² ≠ 0 になった瞬間、 (i+½, j) にいるターゲットが (i, j±½) に住む α₂ を要求する — 半セルずれた別配置の値で、近傍 4 点の平均なしには評価できない。 一般に ⋆ の係数は Gram 小行列式で同次数の全成分を混ぜるため、 このミスマッチは次元・次数が上がるほど広がる。

4.3 それでも staggered でやる 3 つの既知経路

経路内容コスト
平均化 Hodge ⋆ を近傍平均込みの疎行列として定義(非直交格子 FDTD、FIT の material 行列平均化、 mimetic 差分の系譜) ⋆∘⋆ = ±1 の厳密性が落ち、平均化を対称に設計しないと adjoint 構造と エネルギー安定性が崩れる。stencil と halo が拡大
Lebedev 型成分複製格子 全成分を必要な全位置に複製して ⋆ を再び点ごとにする (異方性弾性波の staggered スキームで標準) 自由度が数倍、spurious mode の管理が必要
collocated 化 全成分をセル中心に置く。密な ⋆ も自明に点ごと staggered の利点を放棄(Formurae には既に CollocatedMode がある)

これは構造格子特有の現象ではない。単体的 DEC でも対角 Hodge は circumcentric 双対 + well-centered メッシュという特殊構造への報酬であり、 一般には Whitney 形式の質量行列(非対角)になる。直交性はその「特殊構造」の構造格子版である。

4.4 Formurae での意味 — 契約との衝突

論文の analytic/discrete boundary 段落に明記のとおり、post-fec は 「配置が合わない場所に補間を暗黙に挿入しない」 「resampling が semantically observable なら versioned opaque primitive にする」を設計原則とする。 staggered な form 場への一般 ⋆ はユーザが宣言した場そのものに平均化を作用させるため、 この境界を黙って越えることになる。やるなら上記 3 経路のどれを採るか (平均化ならどのスキーム・どの向き・逆 Hodge の扱い)を仕様化した 新しい opaque contract として明示的に導入する(design/20260713 §9.3 の weighted-adjoint-d graph の一般化が受け皿)。

scalar Δ が先行できる理由はこの裏返しで、lb はソース u が collocated、 face flux は契約内部の補助場に過ぎないため、cross 項の平均化を lb.general@1 の中に閉じ込められ、ユーザに見える placement を一切跨がない。

7. メリット評価 — 対角制限のままで良いか

メリットの実体は「特定の応用クラスを、手書き離散化ではなく宣言的ジオメトリ層 (embedding を書くだけで計量・体積・係数・検証が自動)に乗せられること」である。 現行の例(27 個)はすべて平坦か直交チャートで足りており、 非直交問題には既に回避手段(quoted derivative + 明示 flux の手書き)がある。 つまり「不可能が可能になる」わけではない。

7.1 非対角でないと宣言層に乗らない応用クラス

応用なぜ非対角が必須か直交チャートでの回避
地形追従座標(σ座標)
気象・海洋・地震波		 剪断変換なので誘導計量に交差項 実用上なし(数値的にしか作れない)
cubed-sphere
全球大気コアの標準格子		 gnomonic 写像の座標線が非直交 conformal 版はあるが閉形式なし
graph surface z = f(x,y)
Beltrami flow・膜・minimal surface		 g = I + ∇f∇fᵀ の f_x f_y 項 等温座標は存在するが構成が別問題
交差項つき異方拡散 ∇·(D∇u)
磁化プラズマ熱伝導・異方性多孔質		 D の非対角 = 計量の非対角と同型 格子を主軸に揃えられる場合のみ
ALE・動く格子 / GR 風空間計量 変形・剪断が generic に交差項を生む なし

上の 2 つが重要。地形追従座標と cubed-sphere は Formura 本来のホームドメイン (気象・地球流体)の標準チャートであり、Yin-Yang の次の自然な一歩でもある。 しかも両者とも「直交な代替チャート」が閉形式を持たないため、 記号 embedding を前提とする Formurae のパイプラインでは回避チャート自体が使えない。 非直交性と記号性の要求が二重に衝突しており、一般計量対応なしには このドメインに宣言的ジオメトリで到達する経路がない。

異方拡散は補足つき: 物理的には D は材料テンソルで計量ではないため、 表層で metric として宣言させるのが正しい設計かは別問題。 ただし lb.general@1 の離散機構(対称係数場 Cab、 交差項の保存スキーム)は異方拡散演算子とそのまま共有でき、 実装投資が幾何と材料の両方に効く。

7.2 メリットを過大評価しないための相対化

  • 回避手段の存在。minimal surface の設計文書がそうしたように、 非直交問題は metric 宣言を諦めて明示 flux で手書きすれば現行でも書ける。 一般計量対応の価値は「手書き → 宣言」の格上げ分。 現状は「embedding が唯一の幾何入力」という看板に 「ただし誘導計量が対角になるチャートに限る。直交チャートを見つける微分幾何は ユーザの仕事」という但し書きが付いている状態である。
  • 一般 Laplacian だけでは気象はできない。地形追従や cubed-sphere のドメインには 移流・コリオリ・保存性など他の要素が要る。正確には「チャートのクラスが開く」であって 「応用が完成する」ではない。
  • 検証の頑健性(小さいが実利)。記号的直交性検査は CAS の簡約力に依存しており、 本当は直交なのに証明できず reject される場合がある。general path は その受け皿(hard error → 正しいが遅い一般コード)になる。

8. 推奨 — 判断基準

比較すべきコストはフル一般化ではなく、restricted scope (collocated の scalar Δ = lb.general@1 のみ、staggered 一般 Hodge は据え置き)である。 これは d ≤ 3 の余因子逆行列、√det g の簡約、標準的な 9 点保存スキームだけで済む 既知技術の範囲で、§5 の Phase 4 の本体部分だけに相当する。

判断基準: 地形追従・cubed-sphere・graph surface 系を例ポートフォリオに入れる意志があるなら、 restricted scope はメリットが上回る。入れない(曲面応用は回転面 + Yin-Yang で完結させる)なら、 対角制限の維持で問題ない。その場合は「非直交は明示 flux で書く」を APPLICATIONS.md なりに文書化された公式方針へ格上げしておくと、 論文の limitations の記述(現状のままで整合)とも一貫する。
